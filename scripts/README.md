# Claude Code Scripts

Standalone scripts that enhance Claude Code workflows. These are typically called from hooks, commands, or CLAUDE.md instructions.

## Installation

1. Copy the script to your project's `.claude/scripts/` directory
2. Make it executable: `chmod +x .claude/scripts/script-name.sh`
3. Reference it from hooks or CLAUDE.md
